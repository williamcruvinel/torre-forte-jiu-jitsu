Landing page desenvolvida para a escola de Jiu-Jitsu **Torre Forte**, com foco em **apresentação institucional, captação de novos alunos e experiência moderna de navegação**.

O projeto foi construído utilizando **HTML, CSS e JavaScript puro**, priorizando:

* organização de código
* componentização de estilos
* responsividade
* animações e interações modernas

# 🎯 Objetivo do Projeto

Criar uma **landing page moderna para uma academia de Jiu-Jitsu**, apresentando:

* filosofia da escola
* metodologia de ensino
* horários de aula
* conquistas em competições
* parceiros
* canais de contato

O projeto também foi utilizado como **exercício de boas práticas de organização de CSS e responsividade**.

| Dispositivo | Largura       |
| ----------- | ------------- |
| Desktop     | > 992px       |

* menu mobile com animação
* grids responsivos

# 📂 Estrutura do Projeto

### Estrutura inicial

Na primeira versão do projeto, os estilos estavam centralizados em um único arquivo:

```
torre-forte/
│
├── index.html
├── style.css
├── script.js
│
└── assets/
    ├── image/
    └── css/
```

---

### 🔧 Refatoração de CSS

Posteriormente o projeto foi **refatorado para melhorar organização e escalabilidade**, separando os estilos por seção.

```
assets/
 └── css/
     ├── globals.css
     ├── header.css
     ├── hero.css
     ├── sobre.css
     ├── horarios.css
     ├── parceiros.css
     ├── competicoes.css
     ├── contatos.css
     ├── footer.css
     ├── buttons.css
     ├── animation.css
     └── watsap_float.css
```

### Benefícios da refatoração

* código mais organizado
* manutenção mais simples
* melhor escalabilidade
* separação de responsabilidades
